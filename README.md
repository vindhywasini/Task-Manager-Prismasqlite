Task Management System (Fullstack)
---------------------------------
This archive contains two folders:
- backend : Node.js + TypeScript + Prisma + SQLite (API)
- frontend: Next.js + TypeScript (App Router)

Quick start (backend):
cd backend
npm install
npx prisma generate
npx prisma db push
npm run dev

Quick start (frontend):
cd frontend
npm install
npm run dev

Default ports:
- Backend: 4000
- Frontend: 3000
